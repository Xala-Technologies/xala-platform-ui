# HTML to Platform UI Translation Table

Quick reference for translating HTML intents to Platform UI components.

## Layout Elements

| HTML Intent | Platform UI Component | Import |
|-------------|----------------------|--------|
| `<div class="container">` | `Container` | `@xala-technologies/platform-ui/primitives` |
| `<div style="display:flex">` | `Stack direction="horizontal"` | `@xala-technologies/platform-ui/primitives` |
| `<div style="display:flex; flex-direction:column">` | `Stack direction="vertical"` | `@xala-technologies/platform-ui/primitives` |
| `<div style="display:grid">` | `Grid` | `@xala-technologies/platform-ui/primitives` |
| `<section>` | `Card` with `role="region"` | `@digdir/designsystemet-react` |
| `<article>` | `Card` with `role="article"` | `@digdir/designsystemet-react` |

## Forbidden Patterns

**NEVER use these raw HTML elements. The linter will error:**

```tsx
// ❌ FORBIDDEN
<div>Content</div>
<span>Text</span>
<p>Paragraph</p>
<h1>Title</h1>
<section>Section</section>
<button>Click</button>

// ✅ CORRECT
<Stack>Content</Stack>
<Paragraph>Text</Paragraph>
<Paragraph>Paragraph</Paragraph>
<Heading level={1}>Title</Heading>
<Card role="region">Section</Card>
<Button>Click</Button>
```

## Quick Reference Card

```tsx
// Common import pattern
import { Heading, Paragraph, Button, Card } from '@digdir/designsystemet-react';
import { Stack, Grid, Container } from '@xala-technologies/platform-ui/primitives';
import { DataTable, FormLayout } from '@xala-technologies/platform-ui/composed';
import { EmptyState, AccessGate } from '@xala-technologies/platform-ui/blocks';
import { DashboardLayout } from '@xala-technologies/platform-ui/shells';

// Common composition
<DashboardLayout>
  <Stack direction="vertical" gap="4">
    <Heading level={1}>Page Title</Heading>
    <Card>
      <Paragraph>Content</Paragraph>
    </Card>
  </Stack>
</DashboardLayout>
```
